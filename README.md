# 关于StamonVM
* StamonVM（简称STVM）是一个为了自制编程语言而实现的虚拟机
* Climber-Rong为了实现名为Stamon的自制编程语言，在2023年初开始构思，并在创建了这个项目
* StamonVM通过架构设计，使得其可以运行不止一门编程语言
